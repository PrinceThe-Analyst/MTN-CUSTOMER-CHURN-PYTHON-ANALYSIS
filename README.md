# MTN-CUSTOMER-CHURN-PYTHON-ANALYSIS# 📊 MTN Customer Churn Analysis

## 📌 Project Overview
This project analyzes customer churn behavior for MTN using Python. The goal is to identify key factors driving customer churn, understand revenue loss, and generate insights that can help improve customer retention strategies.

---

## 🎯 Objectives
- Identify the main reasons customers churn
- Calculate total revenue lost due to churn
- Analyze customer behavior based on demographics and usage
- Discover patterns in subscription plans, devices, and satisfaction levels

---

## 🛠 Tools & Libraries Used
- Python 🐍
- Pandas
- NumPy
- Matplotlib

---

## 📂 Dataset Description
The dataset contains customer information such as:
- Customer ID
- Age & Gender
- Location (State)
- Device Type
- Subscription Plan
- Customer Satisfaction Rating
- Tenure
- Monthly Charges
- Total Revenue
- Reason for Churn

---

## 📊 Key Analysis Performed

### 1. Total Revenue Lost Due to Churn
Calculated total revenue lost from all churned customers.

### 2. Top Reasons for Churn
Identified major reasons such as:
- Better offers from competitors
- Poor customer service
- Poor network quality
- High tariffs

### 3. Customer Segmentation Analysis
Analyzed churn patterns based on:
- Device type
- Subscription plans
- Age groups
- States

### 4. Customer Satisfaction Impact
Explored how satisfaction levels relate to churn behavior.

---

## 📈 Key Insights
- Competitor offers are the leading cause of churn
- Certain states experience higher revenue loss than others
- Specific subscription plans are more prone to churn
- Customer satisfaction plays a role in retention
